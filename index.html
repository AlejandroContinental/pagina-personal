Proyectos Académicos

Proyecto Llankay

Aplicación para Windows que permite a los técnicos del Cusco ofrecer sus servicios de manera rápida y segura. Desarrollada con C# y .NET Framework, incluye sistema de calificaciones y geolocalización.

Autonet

Programa hecho en C++ para Ubuntu server que automatiza la configuración de interfaces de red, enrutamiento y servicios DHCP. Ideal para administradores de red que necesitan configurar múltiples equipos rápidamente.

Proyecto VOGUI

Asistente de voz que describe el entorno y objetos para personas con discapacidad visual, desarrollado con Arduino, sensores ultrasónicos y síntesis de voz. Ganador del Hackaton de Innovación 2023.
